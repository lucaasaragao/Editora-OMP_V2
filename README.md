## 📑 Sobre

Site foi desenvolvido para a **Editora UFPB** com propósito de ser uma plataforma de informações para a instituição. O site foi mencionado e compartilhado pela instituição <a scr="https://pkp.sfu.ca/">PKP</a> que é a responsável  por desenvolver software de código aberto (gratuito) que foi utilizado  como espaço de trabalho online para editores,  <a scr="https://pkp.sfu.ca/omp/">Open Monograph Press</a>.

---

## 🚀 Tecnologias utilizadas
